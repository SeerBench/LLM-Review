# 慧眼鉴真 · SeerBench

慧眼鉴真为 LLM API 中转站提供能力鉴定。你交一个第三方端点，我们用一组持续更新的题库实测它的能力档位，对照直连官方 API 的基准给出评测报告。

> **数据真实可靠，是我们最核心的价值所在。**

[`SeerBench/LLM-Review`](https://github.com/SeerBench/LLM-Review) 是慧眼鉴真的对外宣传站——本仓库不包含实测功能，只解释工具在测什么、怎么测、为什么这样测。完整工具住在另一个非公开仓库里。

---

## 我们做什么

中转站是目前现实条件下，很多人使用国际一线 AI 模型的可行方式。慧眼鉴真用 6 类公开能力题集（数学 / 推理 / 语言 / 指令遵循 / 数据分析 / 代码）实测中转站的真实能力档位。

我们持续跟进测试方法、定期轮换题集、抵抗中转站针对性对抗，帮助用户了解他们花钱买到的中转模型到底是什么档位。

## 6 项能力探测

抽题自月度轮换的公开能力题集，每题都有确定性评分器（数学解析、语义匹配、规则验证、Docker 沙箱测试用例等）。每类默认抽 10 题，与公开 leaderboard 同口径。

| 维度 | 题源 | 评分要点 |
|---|---|---|
| 数学 | 竞赛 / 奥赛 / 本科高难度题 | 上游确定性评分器，兼容 `\boxed{}` 与裸答案 |
| 推理 | 多步逻辑、心理状态追踪、撒谎判断、空间几何 | 区分模型实际能力最直接的维度之一 |
| 语言 | 语义关联、文段重排、拼写纠错 | Claude 4.x 整体偏强但仍可分辨档位 |
| 指令遵循 | 多条强约束的写作任务 | Pure-python verifier 自动判定每条约束 |
| 数据分析 | 表格合并 / 重排 / 列类型标注 | Opus 与 Sonnet 在这里有清晰差距 |
| 代码 | 竞赛编程平台真题 | 生成代码进 Docker 沙箱跑测试用例 |

为避免针对性对抗，我们不对外披露具体题源仓库与子分类标签。

## 评分体系

### 能力档位（4 档刻度尺）

横轴为 6 类能力准确率均值，由公开模型锚定：

| 档位 | 阈值 | 公开模型锚点 |
|---|---|---|
| 旗舰 `TIER_FLAG` | ≥ 0.795 | GPT-5.5 (0.85) · GPT-5.4 (0.82) · Opus 4.7 (0.80) |
| 主流 `TIER_MAIN` | ≥ 0.740 | Opus 4.6 (0.79) · Sonnet 4.6 (0.78) · DeepSeek V4 Pro (0.77) |
| 基础 `TIER_BASE` | ≥ 0.650 | Sonnet 4.5 (0.71) · MiniMax M2.7 (0.66) |
| 入门 `TIER_ENTRY` | < 0.650 | Haiku 4.5 (0.65) 及更轻量模型 |

### 声明匹配概率（Claim Match）

每类能力题目的正确率作 Bernoulli 似然，对 6 个 Claude 档位（Opus 4.7 / Opus 4.6 / Sonnet 4.6 / Sonnet 4.5 / Haiku 4.5 / 低于 Haiku）做 softmax 后验，再读出端点声称那一档的概率：

- **一致** `P ≥ 0.50`
- **存疑** `≥ 0.15`
- **偏离** `< 0.15` — 强证据指向降级

## 设计原则

- **只测能力，不测可伪造信号** — 回答风格、自报身份、拒答模式都可以被代理 mock；实际能力不可伪造，所以我们只考核模型解决客观实际问题的能力
- **与官方直连 API 评测对照** — 几百项题库通过官方直连 API 评测构成基线，被测端点直接对照档位
- **确定性自动评分** — 数学解析、Python verifier、Docker 沙箱跑测试用例。不依赖人工主观评判或 LLM 当裁判
- **优雅降级** — 任何一类探针失败（Docker 不可用、模型超时、超过一半样本未完成）会自动从档位后验里剔除，不污染其他维度

## 对抗与防御

中转站经营者一旦知晓本工具，可能采取针对性对抗：抓取并缓存测试题库、记录评测方固定出口 IP 后仅对其提供旗舰模型、识别请求特征后切换"演戏模式"。

我们的针对性措施：

- **题库随机抽取** — 每次测试用随机种子从大题集里抽题，单题被重复命中的概率极低
- **题库定期更新** — 月度轮换，已外泄题目及时退出
- **付费住宅代理出口** — 中转方看到的来源 IP 与普通终端用户难以区分

防御不是完美的，这是一场长期博弈——但**尽最大可能为用户提供有效、可信的评测数据**，是我们的价值所在。

## 我们承诺不做的事

- ✗ **永远不收「认证费」** — 中转站付费贴「已验证」标签的生意我们不接。一旦评测可被钱影响，整个产品价值即被破坏
- ✗ **没有任何「白名单加分」** — 如设置广告位会清晰标明；承诺广告商与所有匿名提交的测试方法完全一致

## 已知局限性

- **小样本方差** — 免费版每类 10 题，σ ≈ 16pt，对相邻档位判别力有限。多次取平均更稳；付费版完整跑数百题单次即可锁定
- **题库污染风险** — 若被测模型训练时见过题集中部分题目，分数会虚高。月度轮换 + 随机抽样降低概率，但完全杜绝训练污染并不现实
- **基线对照差异** — 公开模型基线按 thinking-on 校准，本工具默认 thinking-off → 绝对分被 1-2 档下移，相对排序仍可比较
