创造价值，绝不作恶

我们相信，创造价值，会产生回馈。

关于我们做什么、为什么这么做，以及背后的取舍。

我们做什么

中转站是目前现实条件下，很多人使用国际一线AI模型的一个可行方式。慧眼鉴真提供模型能力的鉴定工具：你交一个第三方中转站，我们用一组持续更新的题库（数学 / 推理 / 语言 / 指令遵循 / 数据分析 / 代码）实测它的能力档位，对照直连官方 API 的基准平分给出评测报告。

持续跟进测试方法、定期轮换题集、避免中转站针对性对抗，帮助用户了解他们买到的中转模型的实际能力档位 —— 数据真实可靠是我们最核心的价值所在。

我们承诺不做的事

  • ✗ 永远不收「认证费」。有API 中转站主动付费贴「已验证」标签 —— 我们不接这种生意。一旦评测结果可以被钱影响，整个产品的价值即被破坏。
  • ✗ 没有任何「白名单加分」。如设置广告位，会清晰标明。承诺广告商与所有匿名提交的测试站点测试方法完全一致。

我们如何避免针对性对抗刷分

被测站点有动机识别我们的流量、然后专门用更高档位的模型回应——这会让评测变成「刷分」。我们一直与这样的技术性手段长期对抗，以下是部分已经实施的措施：

  • • 随机题库——每次从题库随机抽取试题，月度轮换/补充。中转站要长期靠记题、预先训练或缓存答案来作弊，需要付出更高的成本。
  • • 付费代理 IP 保护服务器身份——我们采购了付费的住宅代理出口IP。这是真金白银的额外成本，但能确保被测的中转站无法识别来源 IP，避免基于来源地址，专门提供高阶模型刷分。
  • • 不公开披露探针的可对抗细节——具体题源仓库、子分类标签、评分阈值我们刻意不外宣。对正常使用者意义不大，但披露后中转站对抗的成本立刻降低。（因此慧眼鉴真项目无法开源，如担心 API 密钥安全性，建议使用小额密钥或测后即删。）

我们选择了更高的研发和运维成本：服务端跑探针、定期更新公开题集、付费代理出口。这些是我们对「数据真实可靠」这件事买单的方式。
