评分体系

能力档位 × 声明匹配

评测分两个独立维度：先把端点放上 能力刻度尺 看实测档位，再用 声明匹配概率 判断它是不是端点声称的那个模型。

能力档位（4 档刻度尺）

轴 = 6 类能力准确率均值
旗舰 TIER_FLAG
≥ 0.795
主流 TIER_MAIN
≥ 0.740
基础 TIER_BASE
≥ 0.650
入门 TIER_ENTRY
< 0.650

档位锚定在公开 leaderboard 数据：旗舰 = GPT-5.5 / Opus 4.7；主流 = Opus 4.6 / Sonnet 4.6；基础 = Sonnet 4.5 / MiniMax M2.7；入门 = Haiku 4.5 及以下。

声明匹配（Claim Match Probability）

Bayesian 后验 P(声明模型 | 观测)
一致 match
P ≥ 0.50
存疑 suspicious
≥ 0.15
偏离 mismatch
< 0.15

每类能力题目的正确率作 Bernoulli 似然，对 6 个 Claude 档位（Opus 4.7 / Opus 4.6 / Sonnet 4.6 / Sonnet 4.5 / Haiku 4.5 / 低于 Haiku）做 softmax 后验，再读出端点声称那一档的概率。"偏离" 即强证据指向降级，等价于旧的 DEFINITE_DOWNGRADE。
